Golang API 文档

学习如何在你的 Go 项目中集成和使用 Go Port Rocket 的核心功能

开始使用

要在你的 Go 项目中使用 Go Port Rocket，首先需要安装依赖：

go get github.com/cc11001100/go-port-rocket

然后，你可以导入必要的包：

import (
    "github.com/cc11001100/go-port-rocket/pkg/scanner"
    "github.com/cc11001100/go-port-rocket/pkg/fingerprint"
    "github.com/cc11001100/go-port-rocket/pkg/output"
)

基本的项目结构如下：

  • pkg/scanner - 端口扫描实现
  • pkg/fingerprint - 指纹识别 (服务和操作系统)
  • pkg/output - 输出处理 (格式化和保存)
  • pkg/api - HTTP API 实现
  • pkg/config - 配置管理
  • pkg/utils - 工具函数
  • pkg/logger - 日志处理

扫描器

扫描器是 Go Port Rocket 的核心组件，用于执行端口扫描操作。

创建扫描器

import "github.com/cc11001100/go-port-rocket/pkg/scanner"

// 创建扫描器实例
s, err := scanner.NewScanner(&scanner.Options{
    Target:           "scanme.nmap.org",
    Ports:            "21-25,80,443",
    ScanType:         "tcp",
    Timeout:          "5s",
    Workers:          100,
    ServiceDetection: true,
    OSDetection:      true,
    VersionIntensity: 7,
})
if err != nil {
    log.Fatalf("Failed to create scanner: %v", err)
}

扫描选项

选项 类型 描述 默认值
Target string 扫描目标 (IP或域名) -
Ports string 端口范围 (例如: "80,443" 或 "1-1024") -
ScanType string 扫描类型 ("tcp" 或 "udp") "tcp"
Timeout string 超时时间 (例如: "5s") "5s"
Workers int 工作线程数 100
ServiceDetection bool 启用服务检测 false
OSDetection bool 启用操作系统检测 false
VersionIntensity int 版本检测强度 (0-9) 7
GuessOS bool 推测操作系统 false
LimitOS bool 限制操作系统扫描 false

执行扫描

// 开始扫描
result, err := s.Scan()
if err != nil {
    log.Fatalf("Scan failed: %v", err)
}

// 处理扫描结果
for _, port := range result.Ports {
    fmt.Printf("Port %d/%s: %s\n", port.Port, port.Protocol, port.State)
    if port.Service != "" {
        fmt.Printf("  Service: %s\n", port.Service)
    }
    if port.Version != "" {
        fmt.Printf("  Version: %s\n", port.Version)
    }
}

// 处理操作系统检测结果
if result.OS != nil {
    fmt.Printf("OS: %s %s (%d%%)\n", result.OS.Name, result.OS.Version, result.OS.Confidence)
}

扫描器方法

  • NewScanner(opts *Options) (*Scanner, error)

    创建新的扫描器实例

  • Scan() (*ScanResult, error)

    执行端口扫描并返回结果

  • ScanAsync() (chan *PortResult, chan *ScanResult, chan error)

    异步执行扫描并通过通道返回结果

  • Stop()

    停止正在进行的扫描

服务检测

服务检测模块用于识别开放端口上运行的服务和版本。

使用服务检测

import "github.com/cc11001100/go-port-rocket/pkg/fingerprint"

// 创建服务检测器
detector := fingerprint.NewServiceDetector(&fingerprint.ServiceOptions{
    Intensity: 7,
})

// 对特定端口执行服务检测
service, version, err := detector.Detect("192.168.1.1", 80, "tcp")
if err != nil {
    log.Printf("Service detection failed: %v", err)
} else {
    fmt.Printf("Detected service: %s %s\n", service, version)
}

服务检测选项

选项 类型 描述 默认值
Intensity int 检测强度 (0-9) 7
Timeout string 每次探测的超时时间 "2s"

操作系统检测

操作系统检测模块用于识别目标主机的操作系统类型和版本。

使用操作系统检测

import "github.com/cc11001100/go-port-rocket/pkg/fingerprint"

// 创建操作系统检测器
osDetector := fingerprint.NewOSDetector(&fingerprint.OSOptions{
    GuessOS: true,
    LimitOS: false,
})

// 执行操作系统检测
osInfo, err := osDetector.Detect("192.168.1.1", openPorts)
if err != nil {
    log.Printf("OS detection failed: %v", err)
} else {
    fmt.Printf("Detected OS: %s %s (Confidence: %d%%)\n", 
        osInfo.Name, osInfo.Version, osInfo.Confidence)
}

操作系统检测选项

选项 类型 描述 默认值
GuessOS bool 启用操作系统推测 false
LimitOS bool 限制操作系统扫描范围 false
Timeout string 探测超时时间 "3s"

输出处理

输出处理模块用于格式化和保存扫描结果。

使用输出处理

import "github.com/cc11001100/go-port-rocket/pkg/output"

// 创建输出处理器
outputter, err := output.NewOutputter(&output.Options{
    Format:    "json",
    OutputFile: "result.json",
    Pretty:    true,
})
if err != nil {
    log.Fatalf("Failed to create outputter: %v", err)
}

// 处理并保存扫描结果
err = outputter.Process(scanResult)
if err != nil {
    log.Fatalf("Failed to process output: %v", err)
}

输出处理选项

选项 类型 描述 默认值
Format string 输出格式 (text, json, xml, html) "text"
OutputFile string 输出文件路径 ""
Pretty bool 美化输出 false

支持的输出格式

  • text - 纯文本格式，适合命令行显示
  • json - JSON 格式，适合程序处理
  • xml - XML 格式，适合与其他系统集成
  • html - HTML 格式，适合生成报告

    err = outputter.Process(result)

        log.Fatalf("Failed to process output: %v", err)
    }
}

异步扫描示例

package main

import (
    "fmt"
    "log"
    "time"

    "github.com/cc11001100/go-port-rocket/pkg/scanner"
)

func main() {
    // 创建扫描器
    s, err := scanner.NewScanner(&scanner.Options{
        Target:           "example.com",
        Ports:            "1-1000",
        ScanType:         "tcp",
        Workers:          200,
        ServiceDetection: true,
    })
    if err != nil {
        log.Fatalf("Failed to create scanner: %v", err)
    }

    // 启动异步扫描
    portChan, resultChan, errChan := s.ScanAsync()

    // 处理端口结果
    go func() {
        for port := range portChan {
            fmt.Printf("Port %d/%s: %s", port.Port, port.Protocol, port.State)
            if port.Service != "" {
                fmt.Printf(" (%s %s)", port.Service, port.Version)
            }
            fmt.Println()
        }
    }()

    // 处理错误
    go func() {
        for err := range errChan {
            log.Printf("Error: %v", err)
        }
    }()

    // 获取最终结果
    result := <-resultChan
    fmt.Printf("\nScan completed in %s\n", result.ScanTime)
    fmt.Printf("Found %d open ports\n", len(result.Ports))
    
    // 等待所有处理完成
    time.Sleep(time.Second)
}
