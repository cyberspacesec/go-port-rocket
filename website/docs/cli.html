命令行接口文档

学习如何通过命令行使用 Go Port Rocket 进行端口扫描和网络探测

安装

使用 Go 工具链安装 Go Port Rocket：

go install github.com/cc11001100/go-port-rocket@latest

或者从源码编译：

git clone https://github.com/cc11001100/go-port-rocket.git
cd go-port-rocket
go build -o go-port-rocket

基本使用

Go Port Rocket 的基本命令结构如下：

go-port-rocket [命令] [选项]

主要命令包括：

  • scan 执行端口扫描
  • api 启动 HTTP API 服务
  • mcpscan 使用 MCP 格式执行扫描
  • mcpapi 启动 MCP API 服务

最基本的扫描命令示例：

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp

扫描选项

扫描命令支持多种选项，以下是详细说明：

选项 缩写 描述 默认值
--target -t 扫描目标 (IP地址或域名) -
--ports -p 端口范围 (例如: 80,443 或 1-1024) -
--scan-type -s 扫描类型 (tcp 或 udp) tcp
--timeout - 超时时间 5s
--workers -w 工作线程数 100
--format -f 输出格式 (text, json, xml, html) text
--output -o 输出文件路径 -
--pretty -P 美化输出 false
--os-detection -O 启用操作系统检测 false
--service-detection - 启用服务检测 false
--version-intensity - 版本检测强度 (0-9) 7
--guess-os - 推测操作系统 false
--limit-os - 限制操作系统扫描 false

高级用法

服务检测

启用服务检测功能可以识别运行在开放端口上的服务类型和版本：

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp --service-detection

操作系统检测

启用操作系统检测功能可以识别目标主机的操作系统类型和版本：

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp -O

版本检测强度

调整版本检测的强度级别 (0-9)，值越高，检测越详细但耗时也越长：

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp --service-detection --version-intensity 9

完整扫描

进行全面的扫描，包括端口扫描、服务检测和操作系统检测：

go-port-rocket scan -t 192.168.1.1 -p 1-1024 -s tcp -O --service-detection --version-intensity 9 --guess-os

输出格式

Go Port Rocket 支持多种输出格式，便于后续处理和分析：

文本格式 (默认)

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp -f text

JSON 格式

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp -f json -o result.json --pretty

XML 格式

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp -f xml -o result.xml

HTML 格式

go-port-rocket scan -t 192.168.1.1 -p 80,443 -s tcp -f html -o result.html

示例

扫描特定端口

go-port-rocket scan -t example.com -p 80,443,8080 -s tcp

扫描端口范围

go-port-rocket scan -t 192.168.1.1 -p 1-1000 -s tcp

UDP 扫描

go-port-rocket scan -t 192.168.1.1 -p 53,123,161 -s udp

保存为 JSON 格式

go-port-rocket scan -t scanme.nmap.org -p 1-100 -s tcp -f json -o scan_result.json --pretty

完整扫描示例

go-port-rocket scan -t scanme.nmap.org -p 21-25,80,443 -s tcp --service-detection -O --version-intensity 7 -f html -o scan_result.html
