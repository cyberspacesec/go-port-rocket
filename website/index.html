强大功能

TCP/UDP 扫描

支持 TCP 和 UDP 协议的端口扫描，可指定端口范围或特定端口

服务识别

精确识别运行在目标端口上的服务及其版本信息

操作系统检测

通过网络指纹识别目标主机的操作系统类型和版本

多种输出格式

支持文本、JSON、XML 和 HTML 格式的扫描结果输出

HTTP API

提供 RESTful API，支持远程调用和集成到其他系统

MCP 支持

多行命令处理功能，方便 AI 系统调用和批量任务处理

快速安装

使用 Go 安装

go install github.com/cc11001100/go-port-rocket@latest

从源码编译

git clone https://github.com/cc11001100/go-port-rocket.git
cd go-port-rocket
go build -o go-port-rocket
